# 🏨 Hotel Management System

A Python-based Hotel Management System designed to automate hotel operations such as room booking, restaurant billing, laundry services, gaming services, customer management, and bill generation through a menu-driven interface.

---

## 📖 Project Overview

The Hotel Management System helps hotel staff manage customer records and calculate charges for various hotel services. The system provides a simple console-based interface where users can:

- Register customer details
- Book rooms
- Order food from the restaurant
- Use laundry services
- Access gaming facilities
- Generate a detailed bill

---

## ✨ Features

### 👤 Customer Management
- Store customer information

## ⚙️ Installation

### Step 1: Install Python

Download Python:

https://www.python.org/downloads/

Verify installation:

```bash

```

## ▶️ Running the Project

Navigate to the project directory:

```bash
cd Hotel-Management-System
```

Run the program:

```bash
python Hotelms.py
```
